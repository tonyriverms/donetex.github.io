Compress(Byte[], Int32, ref ByteCompressionMethods)

Compresses a byte array using a specified algorithm.

Declaration
public static byte[] Compress(this byte[] dataToCompress, int startIndex, ref ByteCompressionMethods algorithm)
Parameters
Type Name Description
System.Byte[] dataToCompress

The byte array to compress.
System.Int32 startIndex

The zero-based position in the array where the compression begins.
ByteCompressionMethods algorithm

The algorithm used to compress the array. This method will check the compression rate after the compression is done, and if it is not smaller than 1 (indicating the "compressed" data takes even more space than the original data and the compression is ineffective), the original data will be returned and this argument will be set ByteCompressionMethods.None.
Returns
Type Description
System.Byte[]

The compressed byte array if the compression is successful; or the bytes in the original byte array from startIndex to the end if the compression is ineffective.
