Class StringBuilderCache

Provides two cached System.Text.StringBuilder instances for frequent use of System.Text.StringBuilder class to build short strings (The maximum number of characters is defined by MaxCacheSize.).

Acquire(Int32)

Returns a cached System.Text.StringBuilder instance if capacity is equal to or smaller than the current cache size, or a new System.Text.StringBuilder instance otherwise. The returned instance must be released by either calling Release(StringBuilder) or GetStringAndRelease(StringBuilder).
