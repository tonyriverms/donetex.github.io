Enum SpecialStringSplit

Specifies some special way to split a string.

Namespace: System
Assembly: DoNetExtension.dll
Syntax
public enum SpecialStringSplit

Fields

Name Description
GetASCIIParts

Gets all ASCII segments. For example, "i*" "1ove" "you[" will be extracted from "i*我1ove爱you[你们".
GetNonASCIIParts

Gets all non-ASCII segments. For example, "我" "爱" "你们" will be extracted from "i*我1ove爱you[你们".
SplitASCIIPartsAndNonASCIIChars

Splits into ASCII segments and non-ASCII characters. For example, "i我1ove爱you[你们" will be split into "i" "我" "love" "爱" "you[" "你" "们".
SplitASCIIPartsAndNonASCIIParts

Splits into ASCII segments and non-ASCII segments. For example, "i我1ove爱you[你们" will be split into "i" "我" "love" "爱" "you[" "你们".
