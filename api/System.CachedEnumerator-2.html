Current

Gets the element in the collection at the current position of the enumerator.

Declaration
public T2 Current { get; }
Property Value
Type Description
T2

The current.
Remarks

After an enumerator is created, the MoveNext() method must be called to advance the enumerator to the first element of the collection before reading the value of the Current property; otherwise, Current is undefined and an System.InvalidOperationException will be thrown.

Exceptions
Type Condition
System.InvalidOperationException

Occurs when this property is accessed before MoveNext() is called.
