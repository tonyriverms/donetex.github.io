Bound<T>(T, T, T)

Returns a value neither larger than maximum nor smaller than minimum. If the current value is neither smaller than minimum nor larger than maximum, then the value itself will be returned; if the current value is larger than maximum, then maximum will be returned; if the current value is smaller than minimum, then minimum will be returned.
