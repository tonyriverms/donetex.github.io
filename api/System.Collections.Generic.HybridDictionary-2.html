Class HybridDictionary<TKey, TValue>

Implements System.Collections.Generic.IDictionary<TKey, TValue> using a ListDictionary<TKey, TValue> when the size of the collection is no more than 10, and then switching to the System.Collections.Generic.Dictionary<TKey, TValue> when the collection gets larger.
