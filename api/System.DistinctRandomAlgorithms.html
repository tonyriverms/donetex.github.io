Enum DistinctRandomAlgorithms

Specifies algorithms to generate distinct random values.

Namespace: System
Assembly: DoNetExtension.dll
Syntax
public enum DistinctRandomAlgorithms

Fields

Name Description
Auto

Let the system determine which algorithm should be applied.
SimpleHash

Simply uses a hashtable to prevent repetition, suitable for generating few distinct random values from a wide range.
Swap

Suits when the ratio of the number of generated random values to the number of all possible values is high (typically higher than 20%) and the latter is not large (typically smaller than 1000).
SwapHash

Suits when the ratio of the number of generated random values to the number of all possible values is high (typically higher than 20%) and the latter is not quite small (typically larger than 1000).
