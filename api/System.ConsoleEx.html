GetArg<T>(String[], Int32, Func<String, T>, T, String[])

Gets the argument of type T from one of the console application string arguments, or from console input if the argument is missing.

Declaration
public static T GetArg<T>(this string[] args, int argIdx, Func<string, T> conversion, T defaultArg, params string[] messages)
Parameters
Type Name Description
System.String[] args

The console application arguments.
System.Int32 argIdx

The index of the argument in the args array. If this index exceeds the size of the args array, then it displays the messages line by line in the console window and attempts to read an argument from the console.
System.Func<System.String, T> conversion

The conversion method that converts the string argument to an argument of type T.
T defaultArg

The default argument if the string argument is null or empty or just consists of white spaces.
System.String[] messages

The messages to be displayed line by line on the console window. Note if a message starts with symbol '@', then it is specially treated and a short text "the default is ..." is appended to the end of the message to display the default argument value.
Returns

T

An argument of T.
