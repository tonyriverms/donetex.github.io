Namespace System.Collections.Generic

Classes

ArrayDictionary<TKey, TValue>

Implements System.Collections.Generic.IDictionary<TKey, TValue> and System.Collections.Generic.IList<T> using a TKey array and a TValue array. Recommended for collections that contain 10 items or less.

BiDictionary<TKey, TValue>

Represents a collection of keys and values that are indexed by both keys and values. The two-way indexing is done by synchronizing a key-value dictionary and a value-key dictionary and ensures one-to-one associations between keys and values.

BinaryTree<T>

HybridDictionary<TKey, TValue>

Implements System.Collections.Generic.IDictionary<TKey, TValue> using a ListDictionary<TKey, TValue> when the size of the collection is no more than 10, and then switching to the System.Collections.Generic.Dictionary<TKey, TValue> when the collection gets larger.

ListDictionary<TKey, TValue>

Implements System.Collections.Generic.IDictionary<TKey, TValue> using a linked list. Recommended for collections that typically contain 10 items or less.

MDictionary<TKey, TValue>

Represents a collection of key/value pairs. Different values can be mapped to the same key.

MEnumerator<T>

Supports a simple sequential iteration over multiple generic collections enumerated by their own enumerators. This class can be viewed as an enumerator of enumerators and the initial "M" stands for "multiple", "merged".

MinHeap<T>

A heap data structure for retrieving smallest element in a collection.

MListDictionary<TKey, TValue>

Represents a collection of key/value pairs. Different values can be mapped to the same key. Recommended for collections that typically contain 10 keys or less.

MSortedDictionary<TKey, TValue>

Represents a collection of key/value pairs that are sorted on the key. Different values can be mapped to the same key. NOTE that the values associated with the same key are not sorted.

MSortedList<TKey, TValue>

Represents a collection of key/value pairs that are sorted on the key. Different values can be mapped to the same key. NOTE that the values associated with the same key are not sorted.

ReadOnlyArrayDictionary<TKey, TValue>

Implements System.Collections.Generic.IDictionary<TKey, TValue>, System.Collections.Generic.IReadOnlyDictionary<TKey, TValue> and System.Collections.Generic.IList<T> using a TKey array and a TValue array.

SegmentEnumerator<T>
