Roughly and randomly splits lines from each of an array of System.IO.StreamReader objects into corresponding two System.IO.TextWriter objects. NOTE that this is not an accurate spliting method. Parameters

The split rate. For example, if this value is 0.3, then about 30% of the lines of each reader will be written into the first writer of the corresponding pair in outputs, and the remaining about 70% of the lines will be written into the second writer.
