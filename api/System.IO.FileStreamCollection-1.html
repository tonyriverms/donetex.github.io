Syntax
public class FileStreamCollection<TDictionary> : IDisposable where TDictionary : IDictionary<string, FileStream>, new()
Type Parameters
Name Description
TDictionary

The type of dictionary used to store and retrieve file streams. This dictionary must use strings as keys and accept System.IO.FileStream objects as values.

You will retrieve each file stream by its ID (as a key in the dictionary).
