Class ListDictionary<TKey, TValue>

Implements System.Collections.Generic.IDictionary<TKey, TValue> using a linked list. Recommended for collections that typically contain 10 items or less.

Remarks

Another implementation similar to ListDictionary<TKey, TValue> but with fixed capacity is ArrayDictionary<TKey, TValue>, which implements System.Collections.Generic.IDictionary<TKey, TValue> and System.Collections.Generic.IList<T> using an array of System.Collections.Generic.KeyValuePair<TKey, TValue> objects.
