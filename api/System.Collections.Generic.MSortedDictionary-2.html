Class MSortedDictionary<TKey, TValue>

Represents a collection of key/value pairs that are sorted on the key. Different values can be mapped to the same key. NOTE that the values associated with the same key are not sorted.

Name Description
TKey

The type of keys in this dictionary.
TValue

The type of values in this dictionary.
Remarks

This class inherits from the class and you may directly operate on the underlying linked list. However, NOTE that the this indexer is modified to return a TValue array instead of a linked list.

This class also implements the IMDictionary<TKey, TValue> interface, but it is highly not recommended to operate on the underlying linked list through this interface.
