LoadInt32Arrays(String)

Treat the current string instance as a file path to a text file, and loads the System.Int32 arrays from the text file. The default encoding is System.Text.Encoding.UTF8.

Each line of text in the file represents one array, and substrings separated by spaces are integer elements in the array.

Empty lines are ignored. Multiple white spaces in a line are treated as a single white space.

LoadStringDictionary(String, String)

Treats the current string instance as a file path to a text file, and loads a string-keyed dictionary from the text file. The default encoding is System.Text.Encoding.UTF8.

Each line of text in the file represents a dictionary entry. Each line is first split by the specified delimiter into a string array; The first element in the array is used as the key, while the remaining elements are converted as System.Double values associated with the key.
